# PDFKey Landing Page

PDFKey 產品的官方 Landing Page，這是一個專為自動解密和開啟加密 PDF 檔案設計的 App 展示網站。

## 關於 PDFKey
PDFKey 是一個專為自動解密和開啟加密 PDF 檔案設計的 App。安全儲存密碼，自動嘗試解密，完全離線運作，保障您的隱私安全。應用程式提供以下核心功能：
1. **密碼安全管理**：所有密碼加密儲存於裝置本地，支援搜尋與排序，輕鬆管理您的所有文件密碼。
2. **自動解密開啟**：開啟 PDF 時自動偵測加密狀態，並嘗試使用儲存的密碼解密，省去重複輸入的麻煩。
3. **隱私絕對安全**：完全離線運作，不收集個人資訊，不傳送資料到外部伺服器。您的資料只屬於您。
4. **生物辨識保護**：支援 Face ID、Touch ID 與指紋辨識，保護您的密碼庫與敏感操作。
